========================================
  股票AI助手  新手上手教程
========================================

这是一个本地运行的 AI 股票分析应用：
你在聊天框里问问题，后端自动抓取实时行情/技术面/新闻
后注入到提示词，再让 DeepSeek 大模型回答。

------------------------------------------
一、首次运行需要做什么（只做一次）
------------------------------------------

【1】确认电脑装好 Python 和 Node.js
    打开 PowerShell 输入：
      py --version          应显示 Python 3.10 或更高
      node --version        应显示 v18 或更高
      npm --version         应显示 9.x 或更高

    若没装：
      Python: https://www.python.org/downloads/  （勾选 Add to PATH）
      Node.js: https://nodejs.org/  （选 LTS 版）


【2】安装后端依赖（在 backend 目录里）
    打开 PowerShell，cd 到本项目 backend 目录，执行：
      py -m pip install -r requirements.txt

    这一步会装：flask、flask-cors、openai、akshare、pandas、python-dotenv
    需要几分钟，等它跑完。


【3】安装前端依赖（在 frontend 目录里）
    新开一个 PowerShell 窗口，cd 到本项目 frontend 目录，执行：
      npm install

    会下载 React/Vite/ECharts 等约 200MB 文件到 node_modules，
    需要几分钟，等它跑完。


【4】配置 DeepSeek API Key
    在 backend 目录下新建文件 .env （文件名以点开头），
    用记事本打开，写入下面四行（替换成你自己的 key）：

      DEEPSEEK_API_KEY=sk-你的key填这里
      DEEPSEEK_BASE_URL=https://api.deepseek.com
      DEEPSEEK_MODEL=deepseek-v4-flash
      FLASK_PORT=3001

    模型可选 deepseek-v4-flash（更快）或 deepseek-v4-pro（更准）。
    .env 文件不要提交到 git，里面有你的密钥。


------------------------------------------
二、日常启动（每次想用都这样）
------------------------------------------

【推荐 - 一键启动】
    双击项目根目录下的 start.bat
    只会弹出 2 个窗口：「股票AI-后端」「股票AI-前端」，并自动打开浏览器。
    关闭这两个窗口即可停止服务。

    首次使用前请确保已完成「一、首次运行」里的依赖安装和 .env 配置。


【手动启动 - 需要同时开两个 PowerShell 窗口】

【窗口 1 - 启动后端】
    cd 到 backend 目录，执行：
      py run.py

    成功标志：屏幕显示
      * Running on http://127.0.0.1:3001

    不要关这个窗口，关了后端就停了。


【窗口 2 - 启动前端】
    cd 到 frontend 目录，执行：
      npm run dev

    成功标志：屏幕显示
      ➜  Local:   http://localhost:3000/

    不要关这个窗口。


【打开应用】
    浏览器访问：http://localhost:3000

    （首次启动 chatgpt 风格界面，左侧栏 + 中间聊天 + 右侧股票面板）


------------------------------------------
三、怎么用 - 三种问问题方式
------------------------------------------

方式1：直接说股票代码
    "分析一下600519最近走势"
    "300750现在适合买吗？"
    后端自动识别 6 位代码，注入实时行情、技术分析、最近新闻。

方式2：直接说股票名字
    "贵州茅台最近怎么样？"
    "宁德时代和比亚迪比较哪个更好？"
    后端会扫描股票名列表，找到匹配的名字。

方式3：问通用知识
    "什么是市盈率？"
    "MACD金叉是什么意思？"
    不涉及具体股票时，AI 用自己的知识回答。


------------------------------------------
四、界面功能说明
------------------------------------------

【左侧边栏 - 从上到下】
  · 股票AI助手品牌      点击折叠侧边栏
  · 新对话              开始一个新的聊天
  · 历史会话            点击切换会话，鼠标悬停显示删除按钮
  · 自选股              展示已添加的自选股票实时价格（每30秒刷新）
  · 股票面板按钮        显示/隐藏右侧股票详情面板
  · 亮色/暗色模式       一键切换主题
  · 设置                打开 API Key / 模型设置

【中间聊天区】
  · 欢迎页有 4 个建议问题，点击直接发送
  · 输入框：Enter 发送，Shift+Enter 换行
  · AI 回复支持 Markdown（标题/列表/代码块/表格）

【右侧股票面板（点"股票面板"按钮打开）】
  · 顶部：股票搜索框（输入代码或名字，点下拉选中）
  · 实时行情卡：价格/涨跌/今开昨收/最高最低/成交量/换手率/市盈率
  · 加入自选/取消自选按钮（右上角 +/− 图标）
  · 技术面摘要：均线排列/MACD信号/量能趋势/各 MA 值
  · K线图：日K/周K/月K 切换，带 MA5/10/20/60 均线 + MACD 指标
  · 近期新闻：8 条标题+摘要+来源+时间，点击跳转原文

五、常见问题（FAQ）
------------------------------------------

Q1: 浏览器打开 http://localhost:3000 是空白页？
A:  检查窗口2有没有显示 "ready"，没有就是前端没起来。
    或者按 F12 看浏览器控制台有没有红色错误。

Q2: 聊天框发问后 AI 回答说"无法获取实时数据"？
A:  说明后端没起来，或者 backend/.env 里的 API Key 错了。
    检查窗口1有没有错误日志，确认 sk- 开头的 key 复制完整。

Q3: 股票面板搜索 600519 但没有下拉？
A:  第一次需要等 akshare 加载股票列表（约10秒），稍等再试。

Q4: K线图显示但价格看着不对？
A:  数据来源是东方财富，前复权处理。如有疑问对比同花顺确认。

Q5: 端口 3000 或 3001 被占用怎么办？
A:  关闭其他应用，或者：
    - 前端改端口：编辑 frontend/vite.config.ts 里的 port: 3000
    - 后端改端口：编辑 backend/.env 里的 FLASK_PORT=3001，
      同时 frontend/vite.config.ts 里的 target 也要改


------------------------------------------
六、如何停止
------------------------------------------

回到那两个 PowerShell 窗口，按 Ctrl+C 退出。
关掉窗口也可以，进程会自动结束。


------------------------------------------
七、技术信息（高级用户看）
------------------------------------------

  前端: React 18 + Vite + TailwindCSS + Zustand + ECharts
  后端: Flask + akshare + DeepSeek OpenAI 兼容 SDK
  数据源: 东方财富 push2 接口（直接 HTTP，绕过 akshare 默认 UA 被屏蔽问题）
  新闻源: akshare.stock_news_em
  通信: SSE（Server-Sent Events）流式输出
  存储: 浏览器 localStorage（自选股/设置/会话历史）
