# 股票分析方法论

## 基本面分析框架

### 核心财务指标
- **市盈率 (PE)**：反映估值水平，需结合行业平均值判断
- **市净率 (PB)**：适用于重资产行业
- **ROE（净资产收益率）**：衡量盈利能力，持续 >15% 为优秀
- **营收增长率**：反映成长性
- **毛利率 / 净利率**：反映盈利质量
- **资产负债率**：反映财务安全边际

### 行业分析要点
- 行业生命周期阶段（成长期 / 成熟期 / 衰退期）
- 行业竞争格局（集中度、护城河）
- 政策环境影响
- 上下游产业链关系

## 技术面分析框架

### 均线系统
- MA5 / MA10：短期趋势
- MA20：中期趋势（月线）
- MA60：中长期趋势（季线）
- 均线多头排列（MA5 > MA10 > MA20 > MA60）为强势信号
- 均线空头排列为弱势信号
- 金叉 / 死叉：短期均线上穿 / 下穿长期均线

### MACD 指标
- DIF 上穿 DEA：金叉，买入信号
- DIF 下穿 DEA：死叉，卖出信号
- 红柱放大：多头力量增强
- 绿柱放大：空头力量增强
- 底背离：股价创新低但 MACD 不创新低，反转信号
- 顶背离：股价创新高但 MACD 不创新高，见顶信号

### 成交量分析
- 放量上涨：多头强势
- 缩量上涨：上涨动力不足
- 放量下跌：恐慌性抛售
- 缩量下跌：空头力量减弱
- 地量：可能见底信号
- 天量：可能见顶信号

### K 线形态
- 大阳线：强势上涨
- 大阴线：强势下跌
- 十字星：多空平衡，变盘信号
- 锤子线 / 倒锤子线：反转信号
- 吞没形态：反转信号

## 消息面分析

### 关注要素
- 宏观经济政策（货币政策、财政政策）
- 行业监管政策变化
- 公司重大公告（业绩预告、重大合同、股权变动）
- 机构研报观点
- 北向资金 / 融资融券动向
